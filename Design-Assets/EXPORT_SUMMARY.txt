QUICK START
===========

1. Extract Design-Assets/ to your repo root

2. Copy logo:
   <img src="/Design-Assets/brand/logo.svg" alt="Tapin" height="72" />

3. Import icon component (React):
   import { ReactComponent as IconAdd } from './Design-Assets/icons/icon-add.svg';
   <IconAdd className="icon-24" />

4. Set CSS variables:
   :root {
     --color-primary: #0B76FF;
     --color-text: #111826;
     --space-md: 16px;
   }

5. Link favicon:
   <link rel="icon" href="/Design-Assets/brand/favicon.ico">

6. Optimize assets (optional):
   ./Design-Assets/optimize-assets.sh
