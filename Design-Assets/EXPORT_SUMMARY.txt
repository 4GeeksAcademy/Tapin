ACCESSIBILITY & WCAG COMPLIANCE
================================

✅ All SVGs include:
   • <title> element
   • aria-label attribute
   • Semantic role="img" for icons

✅ Icons:
   • 24px grid minimum (48×48px tap target with padding)
   • Filled & stroke variants for state indication
   • aria-hidden="true" when decorative

✅ Colors:
   • All text meets WCAG AA (4.5:1 minimum)
   • Primary text meets WCAG AAA (7:1+)
   • Color never sole indicator of status

✅ Focus & Interaction:
   • 2px Tapin Blue focus ring
   • No focus indicators removed
   • prefers-reduced-motion support

QUICK START
===========

1. Extract Design-Assets/ to your repo root

2. Copy logo:
   <img src="/Design-Assets/brand/logo.svg" alt="Tapin" height="72" />

3. Import icon component (React):
   import { ReactComponent as IconAdd } from './Design-Assets/icons/icon-add.svg';
   <IconAdd className="icon-24" />

4. Set CSS variables:
   :root {
     --color-primary: #0B76FF;
     --color-text: #111826;
     --space-md: 16px;
   }

5. Link favicon:
   <link rel="icon" href="/Design-Assets/brand/favicon.ico">

6. Optimize assets (optional):
   ./Design-Assets/optimize-assets.sh

USAGE EXAMPLES
